Accessibility Statement

Last updated June 2026

Our commitment

Aloha Escapes is committed to making this website accessible to all users, including those with disabilities. We target conformance with the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA.

WCAG 2.1 AA — Actively maintained

Technical approach

  • Semantic HTML5 landmarks (nav, main, footer, article)
  • ARIA labels on all interactive elements and regions
  • Color contrast ≥ 4.5:1 for body text, ≥ 3:1 for large text
  • Visible keyboard focus indicators throughout
  • Full keyboard navigation including focus-trapped mobile menu with Escape support
  • prefers-reduced-motion respected — all animations and transitions disabled
  • Descriptive alt text on every image, naming the specific place and scene
  • Live region (aria-live) for form submission status announcements
  • Ambient audio player is opt-in and keyboard-operable; never auto-plays

Testing

Automated accessibility tests (axe-core, WCAG 2.1 AA rule set) run on every code change via GitHub Actions across Desktop Chrome and Mobile Safari. Manual testing is performed with VoiceOver on Safari/macOS and keyboard-only navigation in Chrome.

Known issues

No known WCAG 2.1 AA failures at the time of this statement.

Feedback

If you encounter an accessibility barrier on this site, please contact us with "Accessibility" in the subject line. We aim to respond within two business days and resolve confirmed issues within ten business days.
